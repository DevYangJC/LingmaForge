快速开始

在 5 分钟内完成 灵码工坊 的安装并创建你的第一个 AI 应用。灵感一触即发，编码从未如此简单。

1. 安装灵码工坊 CLI

灵码工坊 提供功能强大的命令行工具，帮助你快速创建、开发和部署项目。

1 npm install -g @lingma/forge-cli
提示: 安装完成后，可运行 forge --version 验证是否成功安装。

2. 创建并启动你的第一个项目

使用以下命令创建一个新项目，并启动本地开发服务器。

1
2 forge create my-first-app
cd my-first-app && forge dev

3. 在浏览器中预览

打开浏览器访问 http://localhost:5173，即可看到默认的示例应用。
