### 登录系统方案设计
- 输出文档: outputs/灵码工坊-登录方案设计.md (完整 15 章节, 含前后端架构、API 规范、安全策略、OAuth 方案、时序图)
- 认证架构: JWT 双 token 轮换 (Access Token 30min + Refresh Token 7day)
- 前端核心: Pinia Auth Store + Axios 拦截器(并发请求刷新队列) + Vue Router 路由守卫
- 后端核心: Spring Security + jjwt + BCrypt + Redis(验证码/黑名单/冷却/限流)
- 登录方式: 验证码登录 + 密码登录 + OAuth(GitHub/微信/Google)
- 关键交互: 首页 prompt → 未登录 → 带参跳转 → 登录后自动带入上下文
- 验证码安全: 三层防护(邮箱60s冷却 + IP 10次/小时 + 单码5次尝试上限)
- Token 安全: Refresh Token 轮换 + 登出黑名单 + CSP 防 XSS
